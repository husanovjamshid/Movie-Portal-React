<h2>Movie Portal Site</h2>

This is a movie portal site where users can browse and search for movies. The site features a database of movies, which includes information such as movie titles, genres, ratings, and release dates. Users can search for movies using keywords or browse by genre.

## Features

* Browse movies by genre
* Search movies by keyword
* View movie details, including title, genre, rating, and release date
* Add movies to a favorites list
* Review movies and leave comments

## Usage

<h4> Browse Movies </h4>
To browse movies by genre, click on the "Genres" link in the navigation menu. This will display a list of available genres. Click on a genre to view a list of movies in that genre.

<h4> Search Movies </h4>
To search for movies by keyword, enter a search term in the search bar and click the search button. The search results will display a list of movies that match the search term.

<h4> View Movie Details </h4>
To view the details of a movie, click on the movie title in the movie list. This will take you to a page with detailed information about the movie, including the title, genre, rating, and release date.

<h4> Add Movies to Favorites </h4>
To add a movie to your favorites list, click on the heart icon next to the movie title. You will need to be logged in to add movies to your favorites list.

<h4> Review Movies and Leave Comments </h4>
To leave a review and comment on a movie, click on the "Reviews" tab on the movie details page. Here you can see existing reviews and leave your own review and comment.
